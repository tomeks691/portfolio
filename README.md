# Portfolio — Tomasz Sosnowski

Personal portfolio website built with Astro and Tailwind CSS.

## 🚀 Tech Stack

- [Astro](https://astro.build) — static site generator
- [Tailwind CSS](https://tailwindcss.com) — styling
- Bilingual — Polish / English

## 📦 Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:4321/portfolio](http://localhost:4321/portfolio)

## 🏗️ Build

```bash
npm run build
```

Static files are output to `./dist/`

## 📁 Structure

```
src/
├── components/     # UI components (Hero, Projects, Skills, Contact...)
├── data/           # Projects data
├── i18n/           # Translations (pl/en)
├── layouts/        # Page layouts
└── pages/          # Routes
```
